# 03. 현장 웹앱 설계

> 진입점: [../README.md](../README.md) · MCP 설계: [02-mcp-design.md](02-mcp-design.md)

---

## 1. 목적

현장 기사가 **양손이 바쁘고 소음이 심한 환경**에서도 등급 계산과 지적사항 조치를 빠르게 입력·조회.
프로토타입은 [../index.html](../index.html)에 이미 구현됨 → Next.js 컴포넌트로 발전.

## 2. 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | Next.js (App Router) |
| 소음 감지 | Web Audio API (`AnalyserNode`로 실시간 dB 측정) |
| 음성 입력 | Web Speech API (브라우저 STT) |
| 백엔드 연결 | MCP 서버의 결정론적 계산/매핑 도구 호출 (또는 LLM 게이트웨이 경유) |

## 3. 핵심 UX — 소음 극복 하이브리드

기계실·비상벨·공사 소음으로 음성 인식이 불가능한 상황을 단계적으로 처리:

```
🎤 음성 인식 시도
   └─> 🔊 주변 소음 dB 측정 (Web Audio AnalyserNode)
         ├─ 80dB 미만(양호) ─> 실시간 STT 작동
         └─ 80dB 이상(심함) ─> ⚠️ "소음 감지" 경고 + STT 차단
                                 └─> ⚡ 터치형 퀵 숏컷 화면 전환
                                       └─> 설비별 다빈도 지적사항 원터치 입력
```

### 3.1 소음 감지 경고
마이크 입력 시 실시간 dB 분석. 80dB 이상이면 "현재 구역은 소음 수준이 높아 음성 인식이
불가능합니다" 경고(시각/진동)를 띄우고 STT 모듈 일시 차단.

### 3.2 퀵 숏컷 폴백
음성 불가 시 화면 상단에 **다빈도 지적사항 칩(Chip)** 노출. 1~2번 터치로 입력 완료:
- **자탐설비**: 감지기 단선, 수신기 비상전원 불량, 경종 미명동
- **스프링클러**: 헤드 살수장애(차폐), 압력챔버 누수, 알람밸브 오작동
- **피난설비**: 유도등 축전지 불량, 방화문 폐쇄장애, 방화셔터 적치물

> 숏컷 목록의 우선순위는 **현장 기사 설문 데이터**로 채택·확장 (가장 자주 발견되는 불량 순).

## 4. 화면 구성 (프로토타입 기능)

[index.html](../index.html)에 이미 구현된 두 위젯이 출발점:

1. **소방 등급 & 의무 계산기**
   건물 종류(아파트/빌딩/상가)·층수·연면적·높이 입력 → 소방안전관리자 선임의무 +
   자체점검 의무(작동/종합) + 필수 소방시설 목록 실시간 계산.
   → Next.js화 시 계산 로직을 MCP `assess_building_fire_obligations` 도구로 위임.

2. **지적사항 정비계획 시뮬레이터**
   대표 불량(감지기 단선·스프링클러 차폐·유도등 예비전원 불량·소방펌프 압력미달) 선택/입력 →
   근거 소방법 + 벌칙 + 조치안 매핑 → 공식 서식용 초안(JSON-LD/Markdown) 생성.
   → MCP `map_defect_to_remediation` + `generate_inspection_report` 도구로 위임.

## 5. 디자인

- 컨셉: Dark Cyber Fire Theme (Charcoal 배경 + 오렌지/엠버/크림슨 그라데이션)
- Glassmorphism 카드, 반응형 레이아웃
- 타이포: Outfit / Inter / Noto Sans KR

## 6. 프로토타입 → 운영 전환 시 분리

| 관심사 | 프로토타입(index.html) | 운영(Next.js + MCP) |
|--------|------------------------|----------------------|
| 계산 로직 | 브라우저 JS에 하드코딩 | MCP 결정론적 도구로 위임 (SSoT 일원화) |
| 지적사항 매핑 | 브라우저 JS 룩업 | MCP `map_defect_to_remediation` |
| 법령 근거 | 정적 텍스트 | MCP 법령 번들 조회 |
| 보고서 출력 | 클립보드 텍스트 | MCP 합성 + 서식 export |
